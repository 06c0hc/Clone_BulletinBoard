# 사용한 기술 스택
+ OS : Windows10 64bit
+ Langauge : JAVA SE 8(JDK 8u151), JSP
+ Web Server : Apache Tomcat 8.5.35
+ DBMS : MySQL 5.7.18
+ Framework : Bootstrap 3.3.7
+ Protocol : HTTP(S)
+ S/W architectural pattern : MVC(Model-View-Controller)
